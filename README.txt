Welcome to the Tic-Tac-Toe game! This project is a simple implementation of the popular Tic-Tac-Toe game using Python. It allows a player to take turns marking spaces on a 3x3 grid and aims to determine a winner or a draw between the player and the computer.
Prerequisites:
To run this Tic-Tac-Toe game, you need to have the following installed:

Python 3
Installation
Clone the repository to your local machine or download the source code files.
Open a terminal or command prompt and navigate to the project directory.

To make a move, enter the corresponding cell number on the grid when prompted.

The cell numbering is as follows:
0 | 1 | 2
---------
3 | 4 | 5
---------
6 | 7 | 8


After each move, the game will display the updated grid.

Customization
You can modify the game to suit your preferences or add additional features. Some possible enhancements include:

Implementing an AI opponent: Add an AI player that can play against a human player.
Graphical user interface (GUI): Develop a graphical interface for the game using a library like Pygame or Tkinter.
Keeping score: Track and display the number of wins for each player across multiple rounds.
Validating user input: Add input validation to ensure players enter valid moves.
